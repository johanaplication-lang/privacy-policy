Política de Privacidad

Prestamista Manager
Última actualización: 11 de febrero de 2026

Resumen: Prestamista Manager almacena todos tus datos exclusivamente en tu dispositivo. No recopilamos ni transmitimos tu información a servidores externos ni a terceros.

1. Información General

Prestamista Manager ("la App") es una aplicación de gestión de préstamos personales desarrollada para dispositivos Android. Esta política describe cómo se manejan los datos dentro de la App.

2. Datos que la App Almacena

Para su funcionamiento, la App almacena los siguientes datos en el dispositivo del usuario:

  • Clientes: nombre, teléfono, email, dirección y notas
  • Préstamos: monto, tasa de interés, número de cuotas, frecuencia de pago, estado y notas
  • Cuotas: montos, fechas de vencimiento, estados de pago
  • Pagos: montos, fechas, tipo de pago y notas
  • Preferencias: tema visual y configuración de la App

3. Almacenamiento Local

Todos los datos se almacenan exclusivamente en el dispositivo del usuario utilizando una base de datos SQLite local. La App no utiliza servidores externos ni almacenamiento en la nube.

El código PIN de seguridad, cuando está habilitado, se almacena de forma segura mediante el sistema de almacenamiento seguro del dispositivo (Android Keystore).

4. Permisos de la App

Acceso a Contactos

La App solicita permiso para acceder a los contactos del dispositivo únicamente para permitir al usuario importar el nombre y teléfono de un contacto al crear un nuevo cliente. Este permiso es opcional y la App funciona completamente sin él.

La App no copia, sincroniza ni transmite la lista completa de contactos. Solo accede al contacto individual que el usuario selecciona manualmente.

Acceso a Almacenamiento

La App solicita acceso al almacenamiento del dispositivo para guardar y restaurar copias de seguridad (archivos JSON) cuando el usuario lo solicita explícitamente.

Integración con WhatsApp

La App permite generar mensajes de recordatorio que pueden enviarse manualmente a través de WhatsApp. La App únicamente abre la aplicación de WhatsApp con un mensaje prellenado; el envío del mensaje depende exclusivamente de la acción del usuario.

Prestamista Manager no accede a conversaciones, contactos de WhatsApp ni almacena información relacionada con el uso de dicha aplicación.

5. Copias de Seguridad (Backup)

La App permite al usuario exportar sus datos como un archivo JSON. Este proceso es:

  • Manual: solo se genera cuando el usuario lo solicita
  • Local: el archivo se guarda en el dispositivo o se comparte mediante las opciones del sistema
  • Responsabilidad del usuario: el usuario decide dónde guardar o enviar el archivo de backup

La App no realiza copias de seguridad automáticas ni transmite datos a servidores externos.

6. Datos que NO Recopilamos

Prestamista Manager no realiza ninguna de las siguientes acciones:

  • No recopila datos para publicidad ni marketing
  • No transmite datos a servidores propios ni de terceros
  • No utiliza servicios de rastreo externos
  • No requiere registro de cuenta ni inicio de sesión
  • No accede a la ubicación del dispositivo
  • No accede a la cámara ni al micrófono
  • No comparte datos con terceros

La App puede registrar eventos técnicos de uso de manera local en el dispositivo con fines de mejora interna y diagnóstico. Estos registros no se transmiten a ningún servidor externo y permanecen exclusivamente en el dispositivo del usuario.

7. Seguridad de los Datos

  • Los datos se almacenan en el espacio privado de la aplicación, inaccesible para otras apps
  • Opción de bloqueo por PIN para proteger el acceso a la App
  • El PIN se almacena de forma cifrada usando Android Keystore

8. Eliminación de Datos

  • Eliminar datos individuales: clientes, préstamos y pagos se pueden eliminar desde la App
  • Eliminar todos los datos: desinstalar la App elimina todos los datos almacenados de forma permanente
  • Borrar datos de la App: desde Ajustes del dispositivo > Aplicaciones > Prestamista Manager > Almacenamiento > Borrar datos

9. Menores de Edad

Esta App está diseñada para uso por adultos en el contexto de gestión financiera. No está dirigida a menores de 18 años.

10. Cambios en esta Política

Nos reservamos el derecho de actualizar esta política de privacidad. Los cambios se reflejarán en esta página con la fecha de actualización correspondiente.
